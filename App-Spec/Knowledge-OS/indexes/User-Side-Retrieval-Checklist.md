# User-Side Retrieval Checklist

## Before answering a significant question
- [ ] Identify surface topic
- [ ] Infer likely underlying concern
- [ ] Estimate whether the user wants clarity, comfort, comparison, challenge, or historical depth
- [ ] Check whether tradition/hermeneutic assumptions are explicit or inferred
- [ ] If low confidence, avoid overfitting to a presumed lens
- [ ] Retrieve text + tradition + method + worldview + implication notes
- [ ] Preserve disagreement where real
- [ ] Balance textual, interpretive, communal, and existential truth
- [ ] Log the question event if it created reusable insight
