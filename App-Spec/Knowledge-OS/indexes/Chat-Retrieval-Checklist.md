# Chat Retrieval Checklist

## Before answering a non-trivial Bible question
- [ ] Identify question type (text / doctrine / reception / tradition / controversy / scholarship)
- [ ] Retrieve candidate notes semantically
- [ ] Pull at least one canonical note if available
- [ ] Pull at least one supporting compiled/source/tradition/controversy note if available
- [ ] Separate text from interpretation
- [ ] Separate historical reception from modern scholarship
- [ ] Preserve disagreement where it exists
- [ ] State uncertainty if support is weak
- [ ] Save durable synthesis back to markdown if useful
