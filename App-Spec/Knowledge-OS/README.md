# Knowledge OS

This folder documents the markdown-first knowledge compilation workflow for the Bible app, JobForcast, and No Guilt.

## Folder Roles
- `raw/` — source inputs
- `compiled/` — machine-assisted intermediate notes
- `canonical/` — stable reusable docs
- `generated/` — reports, answers, derived outputs
- `indexes/` — navigation + routing
- `health-check/` — maintenance artifacts

## Core Rule
If a task produces durable knowledge, it should leave behind a file.
