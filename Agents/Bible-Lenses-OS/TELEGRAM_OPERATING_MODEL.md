# Bible Lenses Agent OS — Telegram Operating Model

## Purpose
Use the Telegram group as mission control for the Bible Lenses build.

## What belongs in the group
- task starts
- progress updates
- blockers
- handoffs
- decisions needed
- completions
- summaries of deeper debates

## What does NOT belong in the group
- every internal thought
- long speculative arguments with no output
- duplicate status spam

## Update format
**Agent:** [name]
**Task:** [what you own]
**Status:** [in progress / blocked / done]
**Need:** [decision / help / none]
**Next:** [next action]

## Daily rhythm
- short standup-style update from each active agent
- blocker escalations as needed
- completion notes when a meaningful chunk lands

## Handoff format
[Agent] handing off [artifact/task] to [next agent] — [reason] — [what is done / what remains]
