# Hermeneutics Index

Master index for interpretive frameworks.

## Core families
- literal / plain sense
- historical-grammatical
- patristic figural
- allegorical
- typological
- scholastic
- Reformation
- historical-critical
- canonical
- narrative
- liberationist
- feminist
- postcolonial
- apocalyptic
- rabbinic / Jewish
- evangelical
- Catholic magisterial
- Orthodox patristic retrieval
- Pentecostal / charismatic

## Build rule
Each hermeneutic note should explain:
- what it assumes
- what it highlights
- what it tends to miss
- which beliefs and figures it most reshapes
